Plotting with Vega lite

Vega lite plot test

The dataset i choose to work on is veterans per county data set which was choosen from data.illinois.gov website. This data contains the number of veterans per county who have an Illinois address recorded in the IDVA Veteran Database.

The data measures number of veterans categorized by dependents, gender etc. in each county. In the plot i am trying to show the number of female veterans in each county of illinois. In the vega lite plot i have used bar type mark with tooltip set to true which details when hovered on it. The bar graph will easy replicated the county with highes number of female veterans and bar graph is an easy way to compare between two information.
